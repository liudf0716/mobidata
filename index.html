MBD-R650L 工业级 5G + WiFi 6 + GPS 路由器

MBD-R650L 是一款面向工业现场、车载系统与物联网场景设计的高可靠网络设备， 集 5G 路由器、网关与 WiFi 中继器于一体，基于 OpenWrt 操作系统，支持深度定制与二次开发。

核心能力

5G 双 SIM 冗余

  • 内置 5G 蜂窝通信模组
  • 双 SIM 卡自动切换
  • 断线检测与自动重连

高性能 WiFi 6

  • 2.4GHz + 5GHz 双频并发
  • 2×2 + 3×3 802.11ax
  • 最大 256 并发用户

工业级接口

  • 5 × 千兆以太网口
  • RS232 / RS485
  • USB 3.0 Host

稳定与安全

  • 硬件 + 软件 WatchDog
  • 防浪涌、防静电
  • 宽温宽压设计

网络与软件特性

网络功能

  • NAT / DHCP / PPPoE
  • 端口映射 / DMZ
  • 流量与带宽控制

VPN 支持

  • PPTP / L2TP / IPSec
  • OpenVPN

工业协议

  • Modbus RTU / TCP
  • 转换 MQTT 网关
  • 云平台协议定制

OpenWrt

  • 完整 OpenWrt 系统
  • 支持客户二次开发

智慧能源 / 智慧交通
